dotNetRDF Demos Server

This web application provides endpoints which demonstrate a variety of dotNetRDF's ASP.Net integration features:

Endpoint URL Feature
/demos/leviathan/ A SPARQL 1.1 compliant query endpoint based on the dotNetRDF Leviathan engine with full text indexing extensions and load on demand from the web enabled
/demos/server/ A SPARQL 1.1 compliant server, provides query endpoint at /demos/server/query, an update endpoint at /demos/server/update and treats all other requests in it's URL space as SPARQL graph store protocol requests.
/demos/books/ Demonstrates a SPARQL 1.1 query endpoint which forwards queries to another remote endpoint, in this case the books endpoint at sparql.org
/demos/sampleConfig A Graph endpoint that returns a RDF graph in an appropriate format based on the clients Accept header, this particular graph is actually the graph that specifies the configuration for these demos.
